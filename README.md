This repository contains avionics and supporting software used for rockets built for the **Czech Rocket Challenge (Advanced Division)**.

## Contents
- Flight/avionics firmware  
- Sensor reading and data logging  
- Configuration and calibration utilities  
- Bench-test code and basic documentation

## Tools
- VS Code  
- PlatformIO / Arduino IDE (depending on board)

## Build & Upload
Clone the repository and open it in VS Code (PlatformIO) or the Arduino IDE.  
Select the correct board and port, then build and upload as usual.

## Notes
This code was developed for competition use under real launch constraints.  
Hardware-specific details (pinouts, wiring, power) may differ between builds.
